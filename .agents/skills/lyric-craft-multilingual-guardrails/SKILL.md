# Lyric Craft And Multilingual Guardrails

Use for local lyric source work only.

Method: prefer concrete object plus action over abstract mood. Use small ordinary comparisons, compression, conversational phrasing, image turns, title payoff, and subtitle-ready line lengths.

Reject generic AI filler, blocked glow/dream/vibe/cosmic-destiny language, repeated motifs without scene function, crowded syllables, unsafe imitation, high-school/minor-coded romance, fake provenance, or rights/platform claims.

For multilingual lines, include meaning brief, native line, literal gloss, usage, and native/fluent review requirement before provider use.

Provider use, generated audio, upload, publishing, Content ID, monetization, and platform-safety claims remain blocked.
