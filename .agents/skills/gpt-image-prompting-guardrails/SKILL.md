# GPT Image Prompting Guardrails

Use for source-only visual prompt work in Mellow Longplay.

Read first: `AGENTS.md`, `KNOWLEDGE.md`, `docs/operating-boundary.md`, `mellow-longplay/channel.md`, relevant episode `manifest.json`, `source/visual.md`, and `reviews/visual.md`.

Prompt order: use-case, canvas/aspect, scene, subject, composition, lighting/color, style/medium, text policy, provenance boundary, safety boundary.

Default target for current visual work is original reference-free 16:9 cozy music-cover imagery with clean negative space and no competitor/channel/reference-image inputs.

Blocked unless separately approved: image generation/editing, render/export, proof output, candidate creation, upload, API/browser/provider/account automation, credentials, protected-likeness copying, competitor thumbnails, fake provenance, and positive rights/platform claims.
